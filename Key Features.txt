Key Features
User Registration & Authentication:

Users should be able to sign up, log in, and manage their profiles securely.

Implement authentication to ensure that only registered users can upload, read, and delete books.

Book Uploading:

Allow users to upload books in various formats (PDF, ePub, etc.).

Store book details like title, author, genre, and a brief description.

Book Management:

Users should be able to view their uploaded books, edit book details, and delete books they no longer want.

Reading Interface:

Provide a comfortable reading experience within the app, with features like bookmarks, font adjustment, and night mode.
